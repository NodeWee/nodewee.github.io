* 【FARR (Find And Run Robot) 2】[2015~] 启动器。 可理解为系统“运行”功能的增强版

* 【[Listary](http://www.listary.com/)】[2015~]文件浏览增强
> 提高选择文件夹的效率。尤其是经常会遇到的“打开”和“保存”对话框要选择文件路径的时候，可以自动展开到上一次目录，或快速搜索跳转到所需目录。  

* 【[KeePass]()】[2015~] 密码存储与管理  
> 开源的本地账号密码存储管理工具。不同网站不同密码，降低网站泄密危机

* 【[Everything](http://www.voidtools.com/)】[2013~] 本地文件搜索  
> 建立文件名称索引，按文件名来搜索文件和文件夹，速度超级快。 替代系统自带的搜索功能  ]

* 【[Total Commander](http://www.ghisler.com/)】[2015~] 文件资源管理器
> 基本可以替代 Windows 系统自带的 Explorer 了。 上手略有点小麻烦，但学会使用之后会大大的提高文件操作效率，不仅仅是快速的打开想要的文件目录，但仅此一点就节约了很多时间和鼠标手的劳动强度。  

* 【HoneyView】[201512~]看图软件，<del>【Google Picasa】[2013~201512]<del>

* 【迅雷看看】、【Daum Potplayer】 视频播放器。 <del>【KMPlayer】</del>

* 【foobar2000】 音乐播放器

 * 【[AdBlock](https://chrome.google.com/webstore/detail/adblock/gighmmpiobklfepjocnamgkkbiglidom)】[2014~] Chrome 广告屏蔽插件  
 
 * 【SwitchyOmega】 Chrome 代理管理插件
