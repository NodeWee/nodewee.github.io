QR Code Generator 二维码生成器
===

使用 jQuery 和 HTML5 技术，在本地电脑上生成二维码，不会上传数据到服务器端。  
*现在生成二维码的网站很多，但不少网站会收集上传用户填写的二维码内容信息，隐私无法得到保护。便建立了这么一个本地二维码生成工具的页面方便自己使用，也顺便惠及他人。*
